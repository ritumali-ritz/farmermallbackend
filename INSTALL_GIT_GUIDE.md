## Option 1: Install Git for Windows (Recommended)

### Step 1: Download Git
1. Go to: **https://git-scm.com/download/win**
2. Download will start automatically
3. Run the installer

### Step 2: Install Git
1. Run the downloaded `.exe` file
2. Click **"Next"** through the installation
3. **Keep all default options** (just click Next)
4. Click **"Install"**
5. Wait for installation (2-3 minutes)
6. Click **"Finish"**

### Step 3: Restart Terminal
1. Close your current terminal/command prompt
2. Open a new terminal
3. Test with: `git --version`

---

## Option 2: Use GitHub Desktop (Easier GUI)

### Step 1: Download GitHub Desktop
1. Go to: **https://desktop.github.com/**
2. Download GitHub Desktop
3. Install it

### Step 2: Sign in
1. Open GitHub Desktop
2. Sign in with your GitHub account
3. Create repository from GitHub Desktop

---

## Option 3: Skip GitHub - Deploy Directly to Render

**Alternative:** We can deploy to Render without GitHub by:
1. Creating a ZIP file of your project
2. Uploading it directly to Render

**Note:** This is less convenient but works if you don't want to install Git.
